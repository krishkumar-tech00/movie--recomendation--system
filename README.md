Movie Recommendation System

A simple Movie Recommendation System built using Python and Tkinter.

Features

- Select a movie genre
- Get movie recommendations instantly
- Simple and user-friendly GUI
- Built with Python Tkinter

Technologies Used

- Python
- Tkinter

How to Run

1. Clone the repository
2. Open the project folder
3. Run:

python main.py

- Action
- Comedy
- Sci-Fi
